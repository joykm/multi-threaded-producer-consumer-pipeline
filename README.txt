** Instructions on how to compile and run line_processor.c:
    1.) Compile the line_processor using "gcc -std=gnu99 -pthread -o line_processor line_processor.c"
    2.) Run the line_processor with no input using "./line_processor"
    3.) Run the line_processor with input re-direction using "./line_processor < 'input.txt'" where
        file name is in the same directory as line_processor.
    4.) Run the line_processor with input and output re-direction using "./line_processor < 'input.txt' > 'output.txt'" 
        where input.txt and output.txt is in the same directory as line_processor.
